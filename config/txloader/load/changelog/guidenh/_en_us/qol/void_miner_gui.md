# Void Miner Filtering

Ever found the process of adding specific ores to input buses for filtering your array of <Color id="RED">Void Miners</Color> tedious?

Now it is all able to be done through a single UI within the Void Miner itself!

Opening the new interface, you can toggle each ore from the current dimensions drop pool as you require. This change also adds:
- Toggling between blacklist and whitelist mode without having to use a pesky old screwdriver
- Copying and loading filter configurations using a <Color id="GREEN">Data Stick</Color>
- Searching for, and highlighting ores by name
- Adding or removing __all__ ores via a specific button

> [!NOTE]
> This new UI is fully compatible with existing setups, any ore in an input bus __OR__ selected in the UI are filtered
