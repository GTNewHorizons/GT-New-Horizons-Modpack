# Ore Changes
There are a few nice changes that have been made to how ores generate, and are interacted with by players! They are outlined below
# Ore Stats Page
<FloatingImage src="./assets/vein_stats.png" align="left" width="128" wrap="square">
  <ImageAnnotation>
    The new "Vein Stats" page
  </ImageAnnotation>
</FloatingImage>
The <Color id="GREEN">Vein Stats</Color> page in NEI has had a nice overhaul! This encompasses the "Vein Stats", "Small Ore Stats" and "Underground Fluid Stats" pages.

Along with a fresh new look, hovering over a world containing an ore will show the chance for the vein to generate per ore chunk in that world!

The new page still retains the same information present from the previous design, but now can display more than one vein type within a usual page if your GUI scale allows for it.

Ores from the likes of __BartWorks__ have also been refactored to use the same system. No more odd behaviour depending on which ore you are searching for.

# Ore Generation Changes
- Ores generated by both <Color id="GREEN">Bartworks</Color> and <Color id="RED">GT++</Color> have now been reworked to mimic the way GregTech ore veins generate
- Asteroids are more common, but now smaller to compensate
- New <Color color="#DBF1FD">Ice Asteroids</Color> have been added in the asteroid belts
- Veins are now less likely to be erased by the air
- Some of the missing ore veins have been added to NEI for ease of finding and consistency
