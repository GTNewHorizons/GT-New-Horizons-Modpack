# Aspect Recipe Index
The Thaumcraft NEI add-on mod has been rewritten to be a standalone mod and is now called <Color id="RED">Aspect Recipe Index</Color>! It contains all of the regular features with many new features and improvements.

A (non-extensive) list of these features are as follows:

## Features:
- Better alignment of items, aspects, and backgrounds across the board
- Improved performance across the board
- You can now see the page for every aspect at once in the <Color id="GREEN">Aspect Combination</Color> and <Color id="BLUE">Items Containing Aspect</Color> pages of NEI by clicking __See All__
- A new handler has been added for Wand Crafting recipes, keeping them separate from generic shaped arcane crafting!
- Searching for uses of items that contain essentia now shows the use of that essentia in alchemy and infusion
- Recipes to upgrade <Color id="RED">Runic Shielding</Color> are now visible in the Infusion page
- New configs (disabled by default) to show the names and combination recipes of undiscovered aspects
