# Angelica Changes
For those unaware, <Color id="GREEN">Angelica</Color> is the GTNH Optifine replacement, improving rendering across the board, and allowing in-built compatibility for shaders!

This mod is not new, but there are some changes for 2.9! This does not include the huge amount of effort to generally optimise rendering, or bug fixes

## Compact CTM
This one is for the texture developers out there! You may or may not have been aware, but it previously took <Color id="RED">47</Color> unique textures just to make glass look pretty! Angelica has changed this with built-in support for <Color id="GREEN">Compact CTM</Color>. Now you only need to make 5 files per block, meaning 9x less work, and nearly 10x less file space taken up, all with no loss to visual quality

## Advanced Color Tracing
> [!WARNING]
> This is not yet fully implemented into the pack, and work is still continuing. But hey, what is an update without a premature teaser for the next!
Lots of work is going on in the background to give <Color id="GREEN">Advanced Color Tracing</Color> as default in GTNH. Those of you using something like Euphoria Patches will already be familiar, but $$\text{soon}^{TM}$$ there will be full support for any shaders (or possibly without!)
